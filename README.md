# STUMPY Live Demo

Welcome to the [STUMPY](https://stumpy.readthedocs.io/en/latest/) [live demo](http://stumpy-live-demo.herokuapp.com/)!

# Installing Requirements

```
pip install -r requirements.txt
```

# Running The Demo Locally

```
./run.sh
```
